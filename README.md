# Heart Disease Data Visualization Dashboard

An end-to-end analytics project that uses **Python + MySQL + Tableau + Flask** to analyze heart disease risk factors and present insights in an interactive dashboard.

## Problem Statement
Healthcare teams need a way to quickly identify patterns linked to heart disease risk. This project transforms health records into actionable visual insights focused on age, cholesterol, blood pressure, smoking, and lifestyle indicators.

## Business Requirements
- Ingest a heart disease dataset from CSV.
- Store and query data using MySQL.
- Build Tableau visualizations and an executive dashboard.
- Create a Tableau story for narrative analysis.
- Publish and embed the dashboard in a Flask web app.

## Data Pipeline Architecture
1. **Raw Input**: `data/Heart_new2.csv`
2. **Python ETL**: `data/load_dataset.py` normalizes to `data/heart_disease.csv`
3. **MySQL Storage**: `database/schema.sql` creates `heart_disease_dashboard.heart_disease`
4. **Data Load**: `database/insert_data.py` inserts normalized records
5. **Analytics & BI**: Tableau connects directly to MySQL
6. **Presentation**: Flask app embeds published Tableau dashboard

## Setup Instructions

### 1) Prerequisites
- Python 3.10+
- MySQL Server 8+
- MySQL Workbench
- Tableau Desktop/Public

### 2) Install Python Dependencies
```bash
pip install -r requirements.txt
```

### 3) Prepare Dataset
Use the provided source CSV and create a normalized analytics CSV:

```bash
python data/load_dataset.py --input data/Heart_new2.csv --output data/heart_disease.csv
```

### 4) Create MySQL Schema (SQL Workbench)
1. Open MySQL Workbench.
2. Connect to your MySQL server.
3. Open and execute `database/schema.sql`.

### 5) Insert Data into MySQL
```bash
python database/insert_data.py --csv data/heart_disease.csv --truncate
```

Optional environment variables:
- `MYSQL_HOST` (default: `localhost`)
- `MYSQL_PORT` (default: `3306`)
- `MYSQL_USER` (default: `root`)
- `MYSQL_PASSWORD` (default: `root`)
- `MYSQL_DATABASE` (default: `heart_disease_dashboard`)

### 6) Run SQL Analytics Queries
Execute `database/queries.sql` in MySQL Workbench.

Includes:
- Average cholesterol by age
- Heart disease count by gender
- High risk patients (`chol > 240` and `trestbps > 140`)

## Tableau Integration Steps
1. Open Tableau Desktop.
2. Connect -> **MySQL**.
3. Use:
   - Server: `localhost`
   - Port: `3306`
   - Database: `heart_disease_dashboard`
   - Table: `heart_disease`
4. Build calculated fields:

**Risk Category**
```text
IF [chol] > 240 AND [trestbps] > 140 THEN "High" ELSE "Normal" END
```

**Age Group**
```text
IF [age] < 40 THEN "<40"
ELSEIF [age] >= 40 AND [age] < 50 THEN "40-49"
ELSEIF [age] >= 50 AND [age] < 60 THEN "50-59"
ELSEIF [age] >= 60 AND [age] < 70 THEN "60-69"
ELSE "70+"
END
```

## Flask Web Application
The Flask app serves a webpage that embeds your Tableau dashboard.

### Run Flask
```bash
python flask_app/app.py
```

Open `http://127.0.0.1:5000/`

### Configure Tableau URL
Set environment variable `TABLEAU_DASHBOARD_URL` to your published Tableau dashboard link.

Example (PowerShell):
```powershell
$env:TABLEAU_DASHBOARD_URL="https://public.tableau.com/views/YourWorkbook/MainDashboard?:showVizHome=no"
python flask_app/app.py
```
